# Brand Guide — Shiran Gilad Architecture & Interior Design

**Version:** 1.0  
**Focus:** Clean, Minimalist, High-End Architectural Excellence

---

# 1. Brand Identity

## Core Philosophy
Shiran Gilad provides premium architecture and interior design services. The brand reflects precision, elegance, and a deep understanding of space and light.

## Visual DNA
| Value | Visual Expression |
|-----|----------------|
| **Professional** | Clean lines, balanced grids, structured layouts |
| **Elegant** | Generous whitespace, refined typography, subtle transitions |
| **Bespoke** | Focus on high-quality project imagery and custom architectural details |

---

# 2. Typography

## Font Family
- **Primary:** **Assistant** (Google Fonts)
- **Secondary:** System sans-serif fallback

## Scale & Hierarchy
| Level | Weight | Usage |
|-----|-------|--------|
| **Hero Heading** | 700 (Bold) | Main landing page hooks |
| **Section Heading** | 600 (Semi-Bold) | Page section titles |
| **Subheading** | 500 (Medium) | Card titles, labels |
| **Body Text** | 400 (Regular) | Paragraphs, descriptions |

## Rules
- **RTL First:** All typography must prioritize Hebrew readability.
- **Line Height:** Use a comfortable line-height (1.6 - 1.8) for long descriptions.
- **Letter Spacing:** Subtle tracking for headings to enhance the premium feel.

---

# 3. Color Palette

## Primary Colors
- **Background:** White (`#FFFFFF`) / Light Gray (`#F9FAFB`)
- **Text (Primary):** Deep Charcoal (`#1F2937`)
- **Text (Secondary):** Slate Gray (`#4B5563`)
- **Accent:** Architectural Gold / Bronze or Deep Navy (as specified in Tailwind config)

## Rules
- **High Contrast:** Ensure all text meets accessibility standards against the background.
- **Minimalist:** Use color sparingly to let project photography shine.

---

# 4. Component Standards

## UI Primitives
- **Cards:** Clean borders, subtle shadows, and generous padding.
- **Buttons:** Solid architectural shapes, no excessive rounding (favor small radii or square corners).
- **Inputs:** Minimalist borders with clear focus states.

## Motion
- **Reveals:** Soft fade-ins and subtle slide-ups for project cards.
- **Transitions:** Smooth easing for page navigation and image galleries.
- **Parallax:** Subtle parallax effects for hero backgrounds.

---

# 5. Imagery & Media

## Photography Brief
- **Style:** High-resolution architectural photography.
- **Content:** Wide shots of interiors, close-ups of textures/materials, and floor plans.
- **Optimization:** All images must be optimized (WebP) for fast loading without losing detail.

## Video
- **Hero:** Silent, looping architectural videos for mobile and desktop (`hero-vid-desktop.mov`).
